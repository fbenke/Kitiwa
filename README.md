# Kitiwa API Server
Written in Django

## .env file
This needs to be created on your machine if you want to run locally  

**Example:**
```
ENV=dev
DATABASE_URL=postgres://username:password@host:port/dbname
DEBUG=1
TEMPLATE_DEBUG=0
SECRET_KEY=xxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxx
```

**Required:**

- DATABASE_URL
- SECRET_KEY

**Optional:**

- ENV
- DEBUG
- TEMPLATE_DEBUG
